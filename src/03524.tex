Write the equation relating voltage ($v$) and current ($i$) for a capacitor.  Note: this equation will contain a {\it time-derivative} of either voltage or current (i.e. either $dv \over dt$ or $di \over dt$).

\vskip 30pt

\underbar{file 03524}
%(END_QUESTION)





%(BEGIN_ANSWER)

$$i = C {dv \over dt}$$

%(END_ANSWER)
